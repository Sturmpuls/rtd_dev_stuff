Python einrichten¶

Übersicht.

Python für Windows nutzen¶

Python-Package mit pip aus der Kommandozeile installieren. Hierbei steht -m für „Modul“. Python weiß also, dass es das Kommando and das Modul „pip“ weitergeben soll.

python -m pip install <package-name>

Python in Atom (IDE)¶

  1. Install Atom
  2. Go to Atom -> Settings -> Install, search for atom-ide-ui and install
  3. pip install python-language-server
  4. Go to Atom -> Settings -> Install, search for script and install (afterwards python files can be run with Ctrl + Shift + b)
  5. Go to Atom -> Settings -> Install, search for autocomplete-python and install, choose engine Jedi
  6. Ctrl + Shift + p -> Listed alle verfügbaren Kommandos auf
  7. Ctrl + Shift + Alt + o -> Script: Run Options, ermöglicht z.B. das Starten der virtuellen Python-Umgebung
  8. Ctrl + Shift + Alt + b -> Script: Run With Profile, ermöglicht das nutzen zuvor gespeicherter Profile

Launch Atom: das beste ist, Atom im Verzeichnis mit dem eigenen Projekt zu öffnen. So bekommmt man vollen Zugang zu den Environment-Variablen. Shift + Rechtsklick -> cmd -> "atom ."

Virtualenvironments in Python¶

Pipenv¶

Pipenv managed Pakete auf Projekt-Basis. Außerdem führt es eine Liste über alle zur Zeit installierten Pakete in einem Projekt, die genutzt werden kann, um dieses Projekt zu replizieren.

  1. Check ob neueste Python Version installiert ist python --version
  2. Check ob pip installiet ist pip --version
  3. Installiere Paket- und Umgebungsmanager pip install --user pipenv
  4. Check ob pipenv --version funktioniert, falls nicht füge C:\Users\Flo\AppData\Roaming\Python\Python36\Scripts zur PATH Variablen hinzu
  5. cd path\to\myproject
  6. pipenv install [package]

Venv¶

Integrated into Python 3.6.

  1. python3 -m venv /path/to/new/virtual/environment
  2. Shift + Rechtsklick in Projektordner -> cmd -> ..\Scripts\activate aktiviert die virtuelle Umgebung. Pip install etc. funktionieren
  3. deactivate deaktiviert sie
