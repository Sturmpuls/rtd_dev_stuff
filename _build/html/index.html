class DateTime¶

Datetime class

setDate($year, $month, $day)¶

Set the date.

Parameter:
  • $year (int) – The year.
  • $month (int) – The month.
  • $day (int) – The day.
Rückgabe:

Either false on failure, or the datetime object for method chaining.
setTime($hour, $minute[, $second])¶

Set the time.

Parameter:
  • $hour (int) – The hour
  • $minute (int) – The minute
  • $second (int) – The second
Rückgabe:

Either false on failure, or the datetime object for method chaining.
